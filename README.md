# 2026 冬旅行天氣看板

旅途用的單頁網頁，提供：

- 雙模式切換：`今天總覽`（預設） / `明天總覽`
- 另外可快速查看後續日期：按鈕會顯示具體日期（會隨當天日期連動），也可用日期下拉指定
- 依行程自動顯示對應地點（2026/02/08 - 2026/02/16）
- 地點可手動勾選調整，且可一鍵「還原當日行程」
- 每個地點整天概況（天氣、最高/最低溫、降雨、降雪）
- 天氣狀況以符號搭配文字顯示（如 `☀️ 晴朗`、`☔ 小雨`、`❄️ 中雪`）；每小時顯示百分比但不寫「機率」字樣
- 每個地點有小備註：與「前一日行程地點均溫」相比（較暖/較冷/差不多）
- 每小時區塊分開顯示：
  - 若看今天：從現在到今晚 24:00
  - 若看其他日期：該日 00:00 到 23:00
- 雪場地點額外顯示降雪與雪深欄位
- Tangram 卡片優先顯示，失敗時自動 fallback 斑尾高原資料

## 直接使用

1. 在瀏覽器開啟 `index.html`
2. 或使用本機伺服器（建議）：

```bash
cd /Users/vivichen/code/202602japantravel
python3 -m http.server 8000
```

然後開啟 `http://localhost:8000`

## 資料來源

- Weather API: Open-Meteo Forecast API（免 API Key）
- 雪場官方即時頁連結：
  - https://www.madarao.jp/ski/
  - https://www.tangram.jp/ski/

## 說明

- 雪場「營運狀態 / 纜車開放」目前以官方頁連結為主；前端直接抓官方站資料通常會遇到 CORS 限制。
- 若要在頁面內自動整合雪場營運數據，建議加一個小型後端代理（Cloudflare Worker）做抓取與快取。
